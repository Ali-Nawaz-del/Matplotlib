# Matplotlib

A collection of Jupyter notebooks documenting my journey learning data visualization with **Matplotlib** and **NumPy** in Python.

## 📁 Contents

| Notebook | Description |
|---|---|
| `introduction.ipynb` | Basics of `plt.plot()` — creating a simple line plot from numeric and categorical data, adding titles and axis labels |
| `line_plot.pdf` | Example output: a styled line plot with custom color, markers, and labels |
| `chart_types.ipynb` | Overview of common chart types — bar chart, pie chart, histogram, and scatter plot (including multi-group scatter comparisons) |
| `subplots.ipynb` | Combining multiple plots in one figure using `plt.subplot()` and `plt.subplots()` |
| `save_data.ipynb` | Saving figures to file with `plt.savefig()` (custom DPI, tight bounding box) |
| `vectors.ipynb` | Visualizing 2D vectors with `ax.quiver()` — including vector addition and vector comparisons |

## 🛠️ Topics Covered

- Line plots (`plt.plot`)
- Bar charts (`plt.bar`)
- Pie charts (`plt.pie`)
- Histograms (`plt.hist`)
- Scatter plots (`plt.scatter`)
- Subplots (`plt.subplot`, `plt.subplots`)
- Saving figures (`plt.savefig`)
- Vector visualization with NumPy + `ax.quiver`
- Plot styling: colors, markers, labels, legends, grids, axis limits
